Pendahuluan

MyClockApp ("kami", "aplikasi kami") yang dikembangkan oleh BERKAHAPP4U berkomitmen untuk melindungi privasi pengguna. Kebijakan privasi ini menjelaskan bagaimana aplikasi MyClockApp mengumpulkan, menggunakan, dan melindungi informasi Anda.

Tentang Aplikasi

MyClockApp adalah aplikasi jam digital yang menampilkan waktu saat ini berdasarkan negara atau zona waktu yang dipilih oleh pengguna. Aplikasi ini dirancang untuk memberikan informasi waktu yang akurat dan mudah digunakan.

Informasi yang Kami Kumpulkan

MyClockApp TIDAK mengumpulkan, menyimpan, atau membagikan data pribadi pengguna.

  • ❌ TIDAK mengumpulkan nama, email, atau informasi pribadi apapun
  • ❌ TIDAK memerlukan pendaftaran atau login
  • ❌ TIDAK mengakses kontak, lokasi GPS, atau file pribadi Anda
  • ❌ TIDAK membagikan informasi Anda dengan pihak ketiga
  • ❌ TIDAK menggunakan cookies atau tracking

Data yang Disimpan Secara Lokal

Aplikasi hanya menyimpan preferensi berikut secara lokal di perangkat Anda:

  • Pilihan negara atau zona waktu yang Anda pilih
  • Pengaturan tampilan aplikasi (tema, format jam, dll)

Data ini tersimpan hanya di perangkat Anda dan tidak pernah dikirim ke server kami atau pihak ketiga manapun.

Izin Aplikasi

MyClockApp mungkin meminta izin berikut:

  • Akses Internet: Hanya untuk menyinkronkan waktu yang akurat (opsional)

Aplikasi TIDAK meminta izin lain yang dapat mengakses data pribadi Anda seperti kamera, mikrofon, lokasi, atau penyimpanan.

Keamanan Data

Karena kami tidak mengumpulkan data pribadi, tidak ada data yang dikirim ke server eksternal atau disimpan di cloud. Semua data tetap berada di perangkat Anda dan Anda memiliki kontrol penuh atas data tersebut.

Layanan Pihak Ketiga

MyClockApp TIDAK menggunakan layanan pihak ketiga seperti:

  • Google Analytics atau layanan analitik lainnya
  • Layanan iklan (AdMob, dll)
  • Firebase atau database cloud
  • Social media SDK

Hak Pengguna

Anda memiliki hak penuh untuk:

  • Menghapus semua data aplikasi melalui pengaturan perangkat Anda
  • Menghapus instalasi aplikasi kapan saja
  • Menggunakan aplikasi tanpa memberikan data pribadi apapun

Kebijakan untuk Anak-Anak

MyClockApp aman digunakan untuk semua usia, termasuk anak-anak di bawah 13 tahun, karena aplikasi ini tidak mengumpulkan informasi pribadi apapun.

Perubahan Kebijakan Privasi
